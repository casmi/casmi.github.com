Eclipse

  1. Download casmi.jar http://casmi.github.com/dat/casmi-0.0.1-SNAPSHOT.jar
  2. Install maven to your computer and m2eclipse to your eclipse (You can find this at the Eclipse Marketplace)
  3. Download quickstart.zip which is an eclipse project for quickstart. http://casmi.github.com/dat/quickstart.zip
  4. Download jogl native dynamic libraries of your OS Windows 32 bit http://casmi.github.com/dat/win.zip Windwos 64 bit http://casmi.github.com/dat/win_64.zip Mac OS X (32 bit / 64 bit) http://casmi.github.com/dat/mac.tar.gz Linux 32 bit http://casmi.github.com/dat/linux.tar.gz Linux 64 bit http://casmi.github.com/dat/linux_64.tar.gz
  5. Copy downloaded native dynamic libraries to your system path
  6. Import downloaded quickstart project to your eclipse
  7. Add downloaded jar to your Java build path
