What is casmi?

casmi is a visualization library for Java. casmi supports 2D/3D realtime rendering base on OpenGL. With assembling graphics primitives (line, circle, sphere, etc.), you can easily design scene layouts. Smooth tween animations and scene sequencing functions add rich user interface on your application.

Realtime

Realtime rendering based on OpenGL

Great Many Primitives

Great many primitives (line, rectangle, circle, ellipse cube, sphere, torus, cone etc.)
