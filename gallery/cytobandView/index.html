cytoband view

Abstract

"cytoband view" is a viewer for Human chromosome cytoband (Human Genome, UCSC hg.19). When you point a band, the band nane (like, Chr.8q12.1) will be shown. Color of bands are a little bit changed to make a good show.

Inside of Program

  1. It automatically download cytoBand.txt.gz from UCSC Human annotation site and stored to a local machine cache(CytobandLoader.java).
  2. The data are read and are stored to objects (Cytoband.java).
  3. After that, the data were set to the appropriate position according to bands (using base pairs) (CytobandView.java).
  4. To draw the data, it uses Rect method for each band (CytobandView.java).
  5. It also uses HitRect for testing mouse hit to objects. When mouse is on the objects, it writes messages of the band name and draw stroke of the band(CytobandView.java).
  • For download, it uses casmi.net.HTTP
  • For saving the data for cache, it uses casmi.parser.CSV
  • For drawing bands, it uses casmi.element.Rect
  • For detecting mouse hit, it uses casmi.hit.hitRect
  • For writing messages, it uses casmi.element.Text and casmi.font.Font
