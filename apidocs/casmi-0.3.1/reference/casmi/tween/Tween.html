public static Tween from (Tweenable target, TweenType tweenType, int durationMillis, TweenEquation equation)

Convenience method to create a new reversed interpolation.

You need to set the target values of the interpolation by using one of the ".target()" methods. The interpolation will run from these target values to the current values (retrieved after the delay, if any).

The following lines are equivalent (if pooling has been disabled): 

 Tween.from(myObject, Types.POSITION, 1000, Quad.INOUT).target(50, 70);
 new Tween(myObject, Types.POSITION, 1000, Quad.INOUT).target(50, 70).reverse();
Several options such as delays and callbacks can be added to the tween. This method hides some of the internal optimizations such as object reuse for convenience. However, you can control the creation of the tween by using the classic constructor.

Parameters
target The target of the interpolation.
tweenType The desired type of interpolation.
durationMillis The duration of the interpolation, in milliseconds.
equation The easing equation used during the interpolation.
Returns
  • The generated Tween.
